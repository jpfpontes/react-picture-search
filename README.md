# React Youtube Search

## Description
Search for pictures. This project was made using unsplash API and Axios.

## Usage

You need to setup this `.env` key:
```
REACT_APP_UNSPLASH_ACCESS_KEY=<YOUR_UNSPLASH_API_KEY>
```
